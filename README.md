# a-frame-image-tracking

This project demonstrates the practical use of **image tracking in augmented reality (AR)** using **A-Frame.js** and **MindAR**.  
By scanning one of four predefined images with a mobile device camera, users can experience an interactive AR scene featuring a **dancing avatar** along with additional information displayed in the interface.
